# Korean Coach Android App - Plan

## Product Goal
Build a modern Android app for complete beginners (age 40+, zero Korean) to learn practical basic Korean with highly guided UX.

## Core Experience
- 10-minute daily lessons
- Right-brain friendly memory techniques (visual association, story mnemonics, chunking, repetition)
- Speaking + listening drills
- Real-life scenarios (greetings, ordering food, transport, shopping)

## Tech & Quality Requirements
- Latest stable Android stack (Kotlin, Jetpack Compose, Material 3, Navigation, ViewModel, Room, DataStore)
- Clean architecture with clear layering and extensibility
- Professional-grade UI/UX, not outdated
- Modular code structure suitable for future AI tutor/ASR integration

## Delivery Stages
1. Bootstrap project with architecture + design system
2. Build beginner learning flow (units, lessons, flashcards, quizzes, spaced review)
3. Add pronunciation practice screen scaffold and progress analytics
4. Add sample curriculum/content and polished onboarding
5. Write comprehensive README + PROJECT_STATUS.md
